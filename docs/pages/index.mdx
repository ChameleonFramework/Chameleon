# Chameleon

<Callout emoji="" type="error">
  Chameleon is still in it's development phase and is therefore considered unstable and is subject to breaking API changes.
  We don't currently recommend using it in production, even though [the creators](https://github.com/HyperaOfficial) currently do.
</Callout>

## What is Chameleon?
Chameleon is a framework created to allow developers to easily create Minecraft plugins that work across multiple platforms.

### Supported Platforms
Chameleon plans to eventually support every platform in this list.<br/>
If there's another platform that you think we should support, please create a [feature request](https://github.com/ChameleonFramework/Chameleon/issues/new/choose).
- [x] Bukkit
- [x] BungeeCord
- [ ] Fabric
- [ ] Forge
- [x] Minestom
- [x] Nukkit
- [x] Sponge
- [x] Velocity
